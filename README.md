# AnimeListApp

## Descripción del proyecto

Proyecto de aplicación android para gestionar las series de anime pendientes, vistas y viendo

## Vistas de la aplicación

En la vista principal aparecen todas las series añadidas con la etiqueta de pendiente, terminada o el capitulo actual por el que se está viendo. Se puede filtrar por nombre y el estado. También se puede exportar o importar un fichero con los datos.

Permite añadir nuevos elementos usando el botón de + y rellenando los datos, siempre con un nombre distinto a los existentes. Además manteniendo pulsado sobre un elemento se puede modificar o eliminar el elemento.


<img src="https://github.com/jamv0007/AnimeListApp/assets/84525141/1774b8b0-82de-4e23-a0f3-d5e30a3a38a0" width="300" height="600">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;  <img src="https://github.com/jamv0007/AnimeListApp/assets/84525141/e8dfbb1d-985c-4026-ac09-4dbe8674f2fe" width="300" height="600">

Al acceder al elemento pulsando sobre él muestra las temporadas y se puede filtrar por número de temporada. Además se puede añadir una nueva con el número siguiente al actual y un número de capitulos necesario.

<img src="https://github.com/jamv0007/AnimeListApp/assets/84525141/2736646d-a9c4-441e-a76b-db1dfb47413e" width="300" height="600">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;  <img src="https://github.com/jamv0007/AnimeListApp/assets/84525141/330d62ec-07e4-4d60-b66c-12eab721d4ba" width="300" height="600">

Al mantener pulsada una temporada se puede modificar los capitulos y eliminarla. Si se elimina y no es la última se eliminan las posteriores igulamente. El switch permite marcar la temporada como vista, por lo que sus capitulos se marcan como vistos.

Al pulsar sobre una temporada se observan los capitulos de la temporada. Se pueden filtrar por número y marcarlos como vistos lo que actualiza el actual de la vista principal.
